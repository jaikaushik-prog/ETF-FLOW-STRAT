# ETF-FLOW-STRAT
This script implements the "ETF Flow Pressure" alpha strategy. - Logic: Identifies stocks facing mechanical selling pressure from ETF outflows. - Signal: Bottom 10% of Flow Pressure (Pressure &lt; 0). - Execution: Buy at Close (or Next Open), Hold 20 Days. - Risk Control: Volatility Targeting (15% Annualized).
